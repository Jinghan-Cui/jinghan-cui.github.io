# Bilingual Academic Homepage (Jekyll, GitHub Pages)

Minimal bilingual (中文/English) academic homepage template with:
- **Home** (News, Experience, Interests)
- **Research** (with images)
- **Teaching** (courses, students)
- **Publications** (data-driven)
- **Projects**

## Quick Start
1. Push this repo to GitHub and name it `your_username.github.io`.
2. Settings → Pages → Source: Deploy from a branch (`main` / `/`).
3. Visit `https://your_username.github.io` → auto-redirects to `/zh/`.

### Edit
- `_config.yml`: profile, scholar/ORCID links, default language.
- `/_data/nav.yml`: bilingual navigation.
- `/_data/strings.yml`: bilingual UI strings.
- `/_data/publications.yml`: your publications.
- `/zh` and `/en` pages: content in each language.
- `assets/images/*`: replace research images.
- `assets/css/custom.css`: tweak styles.

No unsupported plugins required for GitHub Pages.
